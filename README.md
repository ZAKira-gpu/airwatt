# AirWatt Website

## Project Setup

This project is built with React, Vite, TailwindCSS, and Framer Motion.

### Prerequisites
- Node.js installed

### Installation
Run the following command to install dependencies:
```bash
npm install
```

### Running Development Server
To start the local development server and test the website:
```bash
npm run dev
```
Then open your browser to `http://localhost:5173`.

### Building for Production
```bash
npm run build
```
